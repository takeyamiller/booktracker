# Project Proposal: Book Tracker

## Overview

**Target Audience:** Personal use — anyone who reads books and wants to stay organized

**Problem:** It's hard to remember what you've read, what you're currently reading, and what you want to read next. Notes apps are messy and don't give you structure.

**Value Proposition:** My app helps readers stay on top of their reading life by giving them one place to track every book by status, add personal ratings and notes, and never forget what to read next.

---

## Feature Scope

### Must-Have (MVP)
1. Add books with title, author, genre, and cover image URL
2. Track each book with a status: Want to Read / Currently Reading / Finished
3. Move books between statuses
4. Add a rating (1–5) and personal notes when a book is finished
5. Record the date a book was finished

### Deferred to v2
- Genre-based book recommendations
- Multi-user support
